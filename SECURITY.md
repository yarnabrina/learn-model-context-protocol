# Security Policy

If you discover any security vulnerabilities in this project, please report them privately as soon as possible. Your help in keeping this repository secure is appreciated!

## Reporting a Vulnerability

**Do not open a public issue** to report a security vulnerability, as this can expose the flaw before it can be fixed.

If you discover a security vulnerability, please report it using the [GitHub Security Advisories](https://github.com/yarnabrina/learn-model-context-protocol/security/advisories) feature.

Thank you for helping keep this project and its users safe!
